GitHubで個人サイトを作る

ファイルをアップロードする

準備するもの

  • htmlファイル
  • GitHubのアカウント

    • まずGitHub上でリポジトリ(repository)を新たに作成します

    右上の+ボタンをクリックして，New repositoryを選択してください

    次にリポジトリの名前を決定します．repositry nameに入力します

    但し必ずこの名前は「index.html」にしましょう．

    下に色々選択肢がありますが後で変更できるので無視してかまいません

    入力したら下のcreate repositoryをクリックします．これでrepositoryが作成されました

    そうしたらrepositoryのメイン画面に入ります．

    その画面のまま，uploding an exsisting fileのボタンをクリックしてください

    choose fileをクリックし，htmlファイルを選択してください(Drag files~でも違いません)

    下のcommit changesをクリックし，ファイルのアップロードは完了です

    サイトを公開する

    メインページに戻り，Settingsのタブをクリックします

    タブ欄に見当たらない場合は右にある三点リーダーをクリックすれば現れます

    横のタブからpagesを選択

    BranchのNoneとなっているボックスをクリックし，Mainを選択します

    そうしたら横に出てくるsaveのボタンをクリックします

    すぐに画面が再読み込みされ，「Your GitHub Pages site is currently being built from the main branch. Learn more.」と表示されます．待ちましょう

    1~2分の後，画面を再読み込みしてみましょう．うまく行っていれば下の表示が現れます．Visit Siteをクリックしてページを見てみましょう!!!

    サイトを編集する

    まぁやり方はいろいろありますが(ローカルファイルで上書き(手作業)(Git))(GitHub上で編集)

    その中でもGitHub上で編集する方法を残しておきます

    codeタブからhtmlを選択，コード部分の右上にあるペンマークをクリック

    書き換えた後，下にあるcommit changesをクリック

    終わりです
